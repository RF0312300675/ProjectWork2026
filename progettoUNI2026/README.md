# Oliva Felice – Sito web dimostrativo

Progetto realizzato **esclusivamente a scopo didattico**.

**DISCLAIMER**
L’azienda, i report, i dati, i fatti e le persone citate sono frutto di pura invenzione e non hanno alcun riferimento alla realtà.

## TECNOLOGIE:
- HTML5
- CSS3
- Bootstrap 5 (CDN)
- Bootstrap Icons (CDN)
- VSC (VISUAL STUDIO CODE)

## Struttura:
- index.html
- style.css
- documents (cartella con i documenti al suo interno in formato PDF)
- img (cartella con le immagini usate nella pagina web)

## Avvio:
apri 'index.html' con il browser
**ATTENZIONE!!**
Bootstrap è caricato tramite CDN, quindi è necessaria una connessione a Internet per il corretto funzionamento degli stili e dei componenti

## Note
- Immagini generate con supporto AI.
- PDF creati dall’autore a scopo didattico.
